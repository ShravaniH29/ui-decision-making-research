# UI Design and Decision-Making Study (Swiggy vs Zomato)

## Overview
This project explores how user interface (UI) design influences decision-making in food delivery applications. The study focuses on comparing two widely used platforms, Swiggy and Zomato, to understand how different design approaches impact user behavior.

The analysis looks at how layout, color, content structure, and visual hierarchy affect how users navigate and make choices.

---

## Research Focus
The study examines:
- How clean vs content-heavy interfaces affect decision speed  
- The role of visual hierarchy in guiding user attention  
- How excessive options can lead to cognitive overload  
- The impact of color and layout on usability and user comfort  

---

## Key Insights
- Structured and minimal interfaces help users make faster and more confident decisions  
- Too many options can overwhelm users and increase decision time  
- Clear visual hierarchy improves navigation and reduces cognitive effort  
- Simpler designs create a more comfortable user experience
